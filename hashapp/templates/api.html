Supported algorithms
MD5, SHA1, SHA224, SHA256, SHA384, SHA512
Hash of the string
GET request to https://gethash.io/api/{algorithm}/{string}
Response: JSON
Example:
Request: https://gethash.io/api/md5/sometext
Response: {"hash": "d41d8cd98f00b204e9800998ecf8427e", "success": true}
Hash of the text
POST request to https://gethash.io/api/{algorithm}/
Send JSON {'text': 'sometext\nanother line text\n\n\n'}
Response: JSON
Example:
Request: https://gethash.io/api/md5/{"text":"asd\nqwe\nzxc\n"}
Response: {"hash": "4ef23dbc13334102765b14c9f5ac92be", "success": true}
Files checksum
Files checksum performs with the JS code on the client side, so you could hash large files without uploading it on server.
